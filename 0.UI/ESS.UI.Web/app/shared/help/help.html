开发流程

                1.首先在domain项目中写聚合根，继承Aggregate类。
                2.然后根据需求写对于的command，如CreateUser，LockUser，继承Command类，并写对应的Event。
                3.写unit test，测试功能是否正确。
                4.写readmodel，根据ui需要返回结果。
                5.再ui中的api项目写对应的api。
                6.再web项目写angular对应的service，controller，html。
                7.再web项目中的/app/route.js中加入路由。
                8.把新加的js加入app.bundle文件中。
                9.再运行程序，在模块定义中配置目录和action对应的字段。
